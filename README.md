# Jared Stout Real Estate Tools App

GitHub-ready static web app / PWA.

## Files

- `index.html` — main app file
- `manifest.json` — PWA app manifest
- `assets/app-icon-192.png` — 192x192 app icon
- `assets/app-icon-512.png` — 512x512 app icon
- `assets/apple-touch-icon.png` — iPhone home screen icon
- `assets/thumbnail.png` — social / GitHub preview thumbnail
- `assets/logo-full.png` — Jared Stout logo
- `assets/jared-headshot.png` — Jared headshot

## How to use on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders from this zip.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. GitHub will give you a public app link.

## Notes

This is a static HTML app. No backend is required.
